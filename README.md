# Asteroid Avoider

Play it at: https://asteroid-avoider.herokuapp.com/

![](screenshots/aagame.PNG)

## Controls

#### Move Left: 
Left Arrow Key and/or "A".

#### Move Right: 
Right Arrow Key and/or "D".

##### Press both "Left + A" or "Right + D" at the same time for faster movement.
